# Sedgwick Algorithms Lectures

Data Structures & Algorithms course based on Sedgwick's "Algorithms, 4th Edition".

## Course Structure
- 12 lectures total (one month course)
- 4 hours per lecture (3 lectures per week)
- Language-agnostic pseudocode examples
